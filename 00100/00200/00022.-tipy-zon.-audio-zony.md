# 00022. Типы зон. Аудио-зоны

Типы зон. Аудио-зоны|В этом уроке Вы узнаете о типах зон и аудио-зонах.|wmysterio|wmysterio||||Рассмотрим опкоды, с помощью которых мы можем установить какие люди и транспорт будут появляться в той или иной зоне. Все известные мне типы описаны. Вот команда, которая указывает тип зоны:

```
0767: set_zone 'MARKST' type 14
```

Так же мы можем указать какой транспорт и люди будут существовать в той или иной зоне. Делается это опкодом:

```
0767: set_zone 'MARKST' popcycle_group_for_peds_and_cars_to 14
```

Вот опкод, который указывает в кокой зоне будут ходить только люди:

```
0874: set_zone 'MARKST' popcycle_group_for_peds_to 14
```

В опкодах:\
'MARKST' — имя зоны ( см. [Имена игровых зон](../../dir/gta\_sa/nazvanie\_zon\_dlja\_gta\_san\_andreas/1-1-0-16/) )\
14 — тип зоны ( см. в [базе данных](../../dir/gta\_sa/tipy\_zon/1-1-0-39/) )
